# SwiftRide Architecture Documentation

Interactive architecture documentation for the **SwiftRide Rideshare Platform**.

## 🔗 View Online

👉 **[View the Architecture Diagram](https://teleios-community.github.io/swiftride-architecture/)**

## What's Included

- **System Overview** — All 7 microservices and their connections
- **Service Documentation** — Detailed descriptions of each service
- **Folder Structure** — Full project directory tree
- **Communication Flow** — Ride request lifecycle sequence diagram
- **Technology Stack** — Languages, frameworks, databases per service
- **Frontend Architecture** — Complete component tree

## Tech Stack

| Service | Language | Framework |
|---|---|---|
| Frontend | TypeScript | Next.js |
| Driver Service | TypeScript | Express.js |
| Rider Service | TypeScript | Express.js |
| Trip Service | Python | Flask |
| Matching Service | Go | net/http |
| Email Service | Python | FastAPI |
